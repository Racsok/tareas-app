# FLUJO DE PRIORIDAD
## Descripción del proyecto
*Descripción*: Gestor de tareas inteligente diseñado para la captura rápida de ideas y la gestión de prioridades mediante una jerarquía visual basada en colores. El sistema permite administrar el ciclo de vida completo de una tarea, optimizando la organización personal mediante la segregación de estados (pendientes vs. completadas)

El usuario puede crear una tarea fácilmente con campos para el título de la tarea, descripción, prioridad, y una fecha esperada de finalización de la tarea.

La tarea creada debe tener la opción de modificarse, completarse y/o eliminarse.

Las tareas sin completar estarán listadas todas en orden de prioridad y distinguidas por colores.

Las tareas completadas se quitaran de la lista y se guardarán en otra lista para tareas completadas.

Las tareas en la lista de completadas se mostrarán en orden de completación, y no tendrán las opciones de modificarse ni eliminarse.

Las tareas completadas tendrán la opción de volver a ser una tarea no completada.
